    User Guidance & Policies

    ✕

    What PhishGuard does

    • Analyzes URLs, webpage/email metadata, and social-engineering signals.
    • Returns a risk score (0–100) and a category: Legitimate / Suspicious / Phishing.
    • Provides evidence-based explanation + recommended action.

    Privacy-first rules

    • No long-term storage of email/webpage content.
    • No external enrichment of user data.
    • Analysis runs only when you click “Analyze”.
    • Minimal data is sent to the LLM (if enabled by your workflow).

    How to respond

    • If Suspicious/Phishing: don’t click links, don’t enter credentials, verify sender via official channels.
    • Inspect domains for look-alikes (typosquatting).
    • Watch for urgency language (e.g., “within 24 hours”).

    Disclaimer

    PhishGuard is a research prototype for academic purposes. It does not replace enterprise-grade security tools.
