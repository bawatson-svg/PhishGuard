# PhishGuard Fix - Quick Instructions

## The Problem
"Could not establish connection" error when clicking Analyze.

## The Solution
Updated version 1.0.1 with better error handling.

## How to Update

1. Go to chrome://extensions
2. Find PhishGuard
3. Click "Remove"
4. Extract the NEW PhishGuard_FIXED.zip
5. Click "Load unpacked"
6. Select the phishguard-fixed folder
7. Done!

## Make Sure
- You're on mail.google.com
- An email is fully loaded and open
- Refresh Gmail page if still having issues

## If It Still Doesn't Work
1. Close all Gmail tabs
2. Open new Gmail tab
3. Wait for email to fully load
4. Click PhishGuard icon → Analyze

The extension now automatically retries if the connection fails.
